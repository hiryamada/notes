# Azure ADのグループ

グループを使用して、ロールやアプリの割り当てを効率的に管理できる。グループは入れ子にできる。

- グループの種別
  - セキュリティ（通常はこちらを利用）
  - Microsoft 365（Microsoft 365のTeamsやSharePointで認識されるグループを作成）
- グループ名
- メンバーシップの種類
  - 割り当て済み
    - 手動でユーザーを指定
  - 動的ユーザー
    - 条件式を指定して、条件に該当するユーザーを自動的にメンバーにする
    - 例: 役職（jobTitle）が「pm」（プロジェクトマネージャ）であるユーザーを pm_group に所属させる
