
# listKeysオペレーションの一覧

「listKeys」: アクセスキー、アカウントキー、ホストキー、接続文字列などを取得するためのアクション

```
$ curl https://docs.microsoft.com/ja-jp/azure/role-based-access-control/resource-provider-operations |grep -i listkey -A 1
```
