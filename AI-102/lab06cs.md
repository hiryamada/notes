# ラボ06 テキストの翻訳

■概要

- 言語サービスを使用します。
- 英語やフランス語で書かれたテキストを読み取り、言語を検出します。また、テキストを英語に翻訳します。

■ヒント

- このラボのコードでは、（SDKではなく）REST APIを使用しています。
- Visual Studio Codeでは、コードを貼り付けた後、Ctrl + Shift + F を使用してフォーマット（インデントを修正）できます。
