# GH-900 GitHub Foundations

https://learn.microsoft.com/ja-jp/collections/66w2a7txpy78r1

- 2日間コース
- GitHubの基礎を学びます
- 本コースではラボ環境（https://esi.learnondemand.net/）の提供はございません
  - GitHubが提供する「演習」（GitHub Skills）をご案内いたします
  - ご自身のGitHubアカウントを利用して「演習」を実施いただけます
  - GitHubの操作画面および演習手順は英語のみとなります。
    - 必要に応じてWebブラウザーの翻訳機能を使用してください。

## 演習

- [各演習の概要](github-skills.md)
- 演習0. GitHubアカウントの準備
  - https://github.com/ にアクセスし、 **個人用のメールアドレス** を使用してサインアップ
  - 以降の「演習」ではこのアカウントを利用してください
- 演習1. GitHub の概要
  - https://github.com/skills/introduction-to-github
- 演習2. GitHub Codespaces
  - https://github.com/skills/code-with-codespaces
- 演習3. Markdownを利用する
  - https://github.com/skills/communicate-using-markdown
- 演習4. プルリクエスト
  - https://github.com/skills/review-pull-requests
- 演習5. GitHub Advanced Securityのシークレットスキャン
  - https://github.com/skills/introduction-to-secret-scanning
- 演習6. GitHub Actions
  - https://github.com/skills/hello-github-actions
