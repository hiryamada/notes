# Azure Bastion

プロビジョニングされる仮想ネットワーク内のすべての VM に対して安全な RDP 接続（RDP connectivity）および SSH 接続（SSH Connectivity）を提供。

■ネイティブ クライアント サポート（Windows ローカル コンピューター上のネイティブ クライアントを使用して Azure のターゲット VM に接続する）
