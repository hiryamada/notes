## github-script

https://github.com/actions/github-script

GitHub APIを簡単に操作したり、ワークフロー内でカスタムロジックを簡単に追加できる。

GitHub Actionsの「アクション」の一種。

特に、GitHubのリソース（Issues、Pull Requests、Actionsの状態など）に対する操作を効率よく行いたい場合に非常に便利


## Octokit

https://github.com/octokit


Octokitは、GitHubの公式クライアントライブラリの一つで、GitHubのリソース（リポジトリ、イシュートラッキング、プルリクエスト、ユーザー管理など）にアクセスするための便利なラッパーを提供する。

## octokit/rest.js

https://octokit.github.io/rest.js/v22/

GitHub REST APIを操作するために用意されたJavaScriptライブラリ

GitHub APIにアクセスするためのHTTPリクエストを行い、そのレスポンスを扱いやすい形式で提供します。これにより、API呼び出しのための複雑なロジックやエラーハンドリングを簡略化できます。


## github-script の中での octokit/rest.js の利用

github-scriptの内部でoctokit/rest.jsを使ってGitHub APIを操作することができます。
