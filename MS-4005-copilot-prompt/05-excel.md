# Microsoft Copilot for Microsoft Excel

## 準備: 自動保存を有効にする

Excelでコパイロットを使用するには、「自動保存」を有効にする必要がある。
![alt text](image-25.png)

「自動保存を有効にする」をクリックし、保存先のOneDriveを選択する。

## 準備: テーブルを作成する
