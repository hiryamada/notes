## コース構成

全6モジュール

- 1. 生成AIとは？ (生成AI, 責任あるAI, Microsoft 365 Copilotの概要)
- 2. AI アシスタントとしてCopilot Chatを採用する (Microsoft 365 Copilot Chat, エージェントの作成)
- 3. ビジネス コンテンツを下書きする (Word, リサーチツールエージェント)
- 4. データを分析および視覚化する (Excel, アナリストエージェント)
- 5. 会議とコラボレーションを管理する (Outlook, Copilotノートブック, Copilot Pages)
- 6. ビジネス ワークフローで Microsoft 365 Copilot が果たす役割を定義する
