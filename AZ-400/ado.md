# Azure DevOps Servicesに含まれるサービス

- Azure Boards - アジャイル計画ツール
- Azure Pipelines - CI/CD
- Azure Repos - プライベートGitリポジトリ
- Azure Test Plans - 手動の探索的テスト
- Azure Artifacts - パッケージ リポジトリ Maven/npm/NuGet/Python 
- 拡張機能のMarketplace
