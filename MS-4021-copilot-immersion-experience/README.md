# MS-4021 Copilot イマージョン エクスペリエンス


## 概要

MS‑4021 は Copilot を業務で活用したいビジネスユーザー向けの短時間・高密度の体験型トレーニングです。

Microsoft 365 Copilot が、財務や人事から営業、運用、マーケティング、IT など、さまざまな機能領域で生産性とイノベーションを強化する方法について説明します

- Microsoft 365 Copilot の実践的な理解を深める
- 自分の業務に Microsoft 365 Copilot をどう活用できるかを考える
- Microsoft 365 Copilot が業務生産性にどう影響するかを体験する

90 分の対面ワークショップ形式
役職別に Copilot のユースケースを体験

コース時間: 90分

## 対象者

Microsoft 365 Copilot を業務で活用したいビジネスユーザー

## 構成

- Microsoft 365 Copilot の基礎理解
- AI 機能、セキュリティ、責任ある AI
- Microsoft 365 アプリとの統合
- 効果的なプロンプト作成
- 高品質なアウトプットを得るためのプロンプト設計
- 各業務シナリオに沿ったデモを実施
