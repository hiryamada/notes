「製品の注文」というイベント発生時に、在庫を更新する処理を行う、C#でコードを開発済み → Azure Functions

リレーショナルDBの運用、オンラインでのトランザクション処理（OLTP） → メモリを多く使用する → メモリ最適化

小規模な処理、イベント発生時の処理ではない、秒単位での処理（秒単位でのコスト支払い） → Azure Container Instance (ACI)
